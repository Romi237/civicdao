# CivicDAO — User Manual

## What is CivicDAO?
CivicDAO is a community governance platform that lets any group
make democratic decisions transparently. Members propose ideas,
vote on them, and track shared finances — all from a mobile app.

---

## Getting Started

### 1. Create your account
1. Open the CivicDAO app on your phone
2. Tap **Sign up** on the login screen
3. Enter your full name, email address, and a password (min 6 characters)
4. Tap **Create account** — you are taken straight to the dashboard

### 2. Log in
1. Open the app
2. Enter your email and password
3. Tap **Sign in**
4. Tap **Forgot password?** if needed

---

## Core Features

### Viewing Proposals
- Tap **Proposals** in the bottom navigation bar
- Use the tabs to filter: **All · Active · Upcoming · Passed · Closed**
- Tap any proposal card to see full details and current vote counts

### Creating a Proposal
1. Tap **+** button in the Proposals screen or **Propose** on the dashboard
2. Fill in:
   - **Title** — short, clear title
   - **Description** — explain the problem and your solution
   - **Budget** — how much funding you are requesting (in USD)
   - **Category** — select the most relevant category
   - **Voting Deadline** — optional, tap the date field to pick one
3. Tap **Submit Proposal**
4. An admin will review and open it for voting

### Voting on a Proposal
1. Open a proposal that shows the **Active** badge
2. Scroll down to the voting section
3. Tap **Vote For** (thumbs up) or **Vote Against** (thumbs down)
4. Confirm your choice — you can only vote once per proposal
5. Results update immediately after your vote

### Treasury
- Tap **Treasury** in the bottom navigation
- See the community balance and recent transactions
- Admins can add deposits and withdrawals

### Members
- Tap **Members** in the bottom navigation
- Search for any member by name or email
- See each member's role and join date

## Admin Features
*(Only visible if you have Admin role)*

### Opening a Proposal for Voting
1. Go to the proposal detail page
2. An admin options menu appears — tap **Open for Voting**
3. The proposal status changes to **Active** and members can now vote

### Managing Members
1. Go to the Members screen
2. Tap the **⋮** menu on any member
3. Choose: **Promote to Admin**, **Suspend Account**, or **Reactivate**

### Adding Treasury Transactions
1. Go to the Treasury screen
2. Tap the **+ Add Transaction** button
3. Choose Deposit or Withdrawal, enter amount and description
4. Tap **Add** to confirm

## Troubleshooting

| Problem | Solution |
|---|---|
| "Cannot reach server" error | Make sure the backend is running and your internet is on |
| Spinning loader that never stops | Pull down to refresh or restart the app |
| "Already voted" message | You can only vote once per proposal — this is by design |
| Admin features not showing | Your account needs admin role — contact your community admin |
| App shows old data | Pull down on any screen to refresh |

---

## Security
- Your password is never stored in plain text
- You are automatically logged out after 7 days of inactivity
- All data is transmitted over HTTPS
